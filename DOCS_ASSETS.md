# Assets ENOTEMPO

## Loghi

Mettere il logo principale in:

- `public/branding/logo-borgogna.jpg`

Altri varianti colore opzionali:

- `public/branding/logo-crema.jpg`
- `public/branding/logo-verde.jpg`

## Evento "Cena di benedizione"

Foto hero:

- `public/events/cena-benizione/hero.jpg`

Foto aggiuntive:

- `public/events/cena-benizione/photo-1.jpg`
- `public/events/cena-benizione/photo-2.jpg`
- ...

## Gallery generale

Tutte le foto della gallery vanno in:

- `public/gallery/`

La pagina `/[locale]/gallery` legge automaticamente
tutti i file `*.jpg` / `*.jpeg` / `*.png` / `*.webp` da questa cartella.
